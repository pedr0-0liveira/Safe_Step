# Safe Step: Tecnologia Assistiva para Mobilidade

O **Safe Step** é um projeto de tecnologia assistiva focado no desenvolvimento de uma bengala inteligente para pessoas com deficiência visual. Utilizando o microcontrolador ESP32 e um sensor ultrassônico como principais componentes, o dispositivo é capaz de detectar obstáculos suspensos em tempo real, proporcionando maior segurança e autonomia ao usuário.

Este trabalho é fundamentado no Trabalho de Conclusão de Curso (TCC) de Rafael Souza (disponível na pasta `docs`). O objetivo principal do Safe Step é otimizar o custo de produção e aprimorar a versão original apresentada pelo autor, buscando maior acessibilidade e eficiência técnica.

## Estrutura do Repositório

* **`/docs`**: Contém o referencial teórico que serviu de base para o projeto, incluindo o TCC de Rafael Souza.
* **`/hardware`**: Esquemas de circuitos, diagramas de conexão e a descrição técnica detalhada dos componentes utilizados (ESP32, Sensor Ultrassônico, etc.).
* **`/media`**: Galeria com fotos do projeto pronto e protótipos.
* **`/src`**: Código-fonte desenvolvido para o funcionamento do projeto no ESP32.

**English**

# Safe Step: Assistive Technology for Mobility

**Safe Step** is an assistive technology project focused on developing a smart cane for individuals with visual impairments. Using the ESP32 microcontroller and an ultrasonic sensor as its main components, the device detects suspended obstacles in real-time, providing increased safety and autonomy for the user.

This project is based on the final thesis (TCC) by Rafael Souza (located in the `docs` folder). The main goal of Safe Step is to reduce production costs and improve upon the version originally presented by Rafael, aiming for greater accessibility and technical efficiency.

## Repository Structure

* **`/docs`**: Contains the theoretical framework used as a basis for the project, including Rafael Souza's thesis.
* **`/hardware`**: Circuit schematics, wiring diagrams, and detailed technical descriptions of the components used (ESP32, Ultrasonic Sensor, etc.).
* **`/media`**: Gallery featuring photos of the completed project and prototypes.
* **`/src`**: Source code developed for the project's operation on the ESP32.
